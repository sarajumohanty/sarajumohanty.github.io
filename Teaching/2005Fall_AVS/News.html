DOWNLOAD AND SETUP INSTRUCTIONS
******************************************************
1. Download the profile_flies.tar and the eng-cadsun.run file into your home directory.
2. Run the following command to add execute permissions to the eng-cadsun.run script.
chmod u+x eng-cadsun.run
3. Now just run the setup script, it will do the job for you and launch icfb!
./eng-cadsun.run
