Course Syllabus and Description

Back to Main Page

Course objectives: Build foundation on digital computer design.

Prerequisites: CSCE 2050

Level of the Course: The course is designed for first year undergraduate students.

Text Book: Computer Organization and Design: The Hardware/Software Interface by D. Patterson and J. Hennessy, Morgan Kaufmann Publishing Co. (4th Edition)

Selected Topics:

  1. Principles of Organization
  2. Instruction Sets
  3. Computer Arithmetic
  4. Datapath and Control
  5. Memory Hierarchies
